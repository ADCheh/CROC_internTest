Для создания новых сценариев в контекстном меню Create необходимо выбрать скриптовый объект Scenario

Данный объект имеет следующие поля:

Title - название сценария

Description - описание сценария

Actions - набор действий в сценарии. Размер поля определяет общее количество действий в сценарии. Элементы добавляются в соответствии с последовательностью действий в сценарии.
Каждый элемент может быть либо кнопкой, либо вентилем.За это отвечает переменная isButton. 
Каждый интерактивынй объект на сцене имеет идентификатор соответсвующий номеру префаба этого объекта. Идентификатор в поле id 
должен соответствовать идентификатору префаба, который находится в папке Prefabs/Buttons или Prefabs/Wheels соответственно.
(Т.е. если необходимо создать действие "Нажмите на кнопку №3", то нужно вы поле id вписать 3, а в поле ObjectFromScene добавить префаб Button_3 из папки prefabs/Buttons.)

Stages - размер этого поля определяет количество этапов в сценарии, а значение каждого элемента определяет количество действий в этапе.
(Т.е. если у нас есть сценарий из 7 действий, который должен быть разбит на 2 этапа, в первом из которыйх 3 действия, а во втором 4, то поле Size будет иметь значение 2,
первый элемент - значение 3, а второй элемент - значение 4).
